# SOD-DETR Benchmark Results

군 경계 작전 환경 데이터셋(AIHub ID 71858 + 71856)에서의 벤치마크 결과이다.
10개 클래스(Fishing_Boat, Merchant_Ship, Warship, Person, Bird, Fixed_Wing, Rotary_Wing, UAV, Leaflet, Trash_Bomb), 테스트 영상 20,172건, GT 인스턴스 41,472건으로 구성된다.

## 시험 환경

| 항목 | 사양 |
|------|------|
| GPU | NVIDIA GeForce RTX 5090 32GB |
| PyTorch | 2.10.0+cu128 |
| CUDA | 12.8 |
| 속도 측정 | warmup 50, 200 runs × 10 rounds, IQR 1.5x trimming, median of means |
| YOLO 평가 | ultralytics.val |
| RF-DETR 평가 | pycocotools (conf threshold 0.01) |
| 온디바이스 | NVIDIA Jetson AGX Orin 64GB, TensorRT FP16 |
